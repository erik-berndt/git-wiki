# Add and Commit

Add single files:  
**$git add filename1 filename2 ...**  

Add all:  
**$git add .**  

Commit:  
**$ git commit -m 'comment'**  

Write comment in editor:  
**$ git commit**

Add all and commit:  
**$ git commit -a -m 'comment'**
